- **多种选择器** - 优先选择内容质量最高的页面区域
- **智能过滤** - 自动移除导航、广告、侧边栏等无关内容

- **页面类型检测** - 自动识别新闻、技术文档、学术论文等类型

### 2. 大模型API集成
- **多服务商支持** - OpenAI GPT、Anthropic Claude、自定义API
- **错误处理** - 完善的API调用错误处理和重试机制
- **响应优化** - 根据不同API格式优化请求和响应处理
- **使用统计** - 记录API使用情况和Token消耗

### 3. 智能提示词优化
- **类型适配** - 针对不同页面类型提供专门优化的提示词
- **质量控制** - 引导AI生成客观、准确的总结内容
- **长度控制** - 根据用户偏好控制总结长度和详细程度
- **效果评估** - 对生成结果进行多维度质量评估

### 4. 用户体验优化
- **快捷键支持** - Ctrl+Shift+S快速总结，Ctrl+Shift+O打开设置

- **历史记录** - 保存所有总结记录，支持查看、删除、导出
- **结果导出** - 支持复制到剪贴板、导出为文本文件

### 5. 隐私与安全
- **本地存储** - 所有敏感信息仅存储在用户本地
- **加密保护** - API密钥使用简单加密存储
- **无数据上传** - 页面内容和总结结果不会上传到第三方
- **权限控制** - 仅请求必要的浏览器权限
