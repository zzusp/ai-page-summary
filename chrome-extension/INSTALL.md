# 安装指南 - 智能页面总结器

本指南将帮助您完成Chrome扩展的安装和配置。

## 📋 系统要求

- **浏览器**: Google Chrome 88+ 或 Microsoft Edge 88+
- **操作系统**: Windows 10+, macOS 10.14+, Linux
- **网络**: 稳定的互联网连接（用于API调用）
- **API账户**: OpenAI或Anthropic账户

## 🚀 安装步骤

### 第一步：准备图标文件

1. **使用现有SVG图标**
   - 扩展已包含 `icons/icon.svg` 文件
   - SVG文件支持多尺寸显示，无需额外准备PNG文件
   - 如果需要更佳显示效果，可以自行创建PNG文件替换

### 第二步：加载扩展

1. **打开Chrome扩展管理页面**
   - 在地址栏输入：`chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

2. **启用开发者模式**
   - 页面右上角找到"开发者模式"开关
   - 点击开关启用

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择包含所有文件的文件夹
   - 点击"选择文件夹"

4. **验证安装**
   - 扩展应该出现在扩展列表中
   - 浏览器工具栏会显示扩展图标
   - 状态应显示为"已启用"

### 第三步：获取API密钥

#### OpenAI API密钥
1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册或登录账户
3. 前往 API Keys 页面
4. 点击 "Create new secret key"
5. 复制生成的密钥（以 `sk-` 开头）

#### Anthropic API密钥
1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册或登录账户
3. 在API Keys部分创建新密钥
4. 复制生成的密钥

### 第四步：配置扩展

1. **打开扩展设置**
   - 点击浏览器工具栏中的扩展图标
   - 在弹出窗口中点击设置按钮（⚙️）

2. **配置API信息**
   ```
   API提供商: 选择你的服务商
   API密钥: 粘贴你的API密钥
   API地址: 可留空使用默认值
   模型名称: 选择推荐的模型
   ```

3. **调整高级设置**
   ```
   创造性参数: 0.3 (推荐)
   最大Token数: 1500
   总结长度: 中等
   ```

4. **测试API连接**
   - 点击"测试 API"按钮
   - 等待测试完成
   - 确认显示"API测试成功"

5. **保存设置**
   - 点击"保存设置"按钮
   - 确认看到"设置已保存"提示

### 第五步：开始使用

1. **首次使用**
   - 打开任意网页
   - 点击扩展图标
   - 点击"开始总结"按钮
   - 等待AI分析完成

2. **验证功能**
   - 确认总结内容显示正确
   - 检查历史记录功能
   - 测试快捷键操作

## ⚠️ 常见安装问题

### 问题1：扩展无法加载
**症状**: "无效的扩展程序"错误
**解决方案**:
```bash
# 检查文件结构
chrome-extension/
├── manifest.json ✓
├── popup.html ✓
├── popup.js ✓
├── options.html ✓
├── options.js ✓
├── content.js ✓
├── background.js ✓
├── styles/ ✓
├── js/ ✓
└── icons/ ✓
```

### 问题2：图标文件缺失
**症状**: 扩展图标显示为灰色
**解决方案**:
```bash
# 确保图标文件存在且格式正确
ls icons/
# 应该包含: icon16.png, icon32.png, icon48.png, icon128.png
```

### 问题3：API测试失败
**症状**: "API测试失败"错误
**解决方案**:
1. 检查API密钥格式是否正确
2. 确认网络连接正常
3. 验证API账户余额
4. 尝试使用不同的模型

### 问题4：权限被拒绝
**症状**: "权限不足"错误
**解决方案**:
1. 确认已启用开发者模式
2. 重新加载扩展
3. 检查Chrome安全设置

## 🔧 故障排除

### 查看控制台日志
1. 右键扩展图标
2. 选择"检查弹出内容"
3. 查看Console标签的错误信息

### 重新安装扩展
1. 在扩展管理页面移除扩展
2. 删除本地文件夹
3. 重新按照安装步骤操作

### 清除缓存数据
1. 打开Chrome设置
2. 隐私设置和安全性
3. 清除浏览数据
4. 清除扩展相关数据

## 📱 移动设备支持

目前此扩展主要设计用于桌面Chrome浏览器。移动设备使用需要：
- 桌面版Chrome浏览器
- 开发者模式支持
- 键盘快捷键（外接键盘）

## 🔄 更新扩展

### 自动更新（推荐发布后）
- 扩展将通过Chrome Web Store自动更新

### 手动更新
1. 下载新版本文件
2. 在扩展管理页面点击"重新加载"
3. 或先删除再重新加载
