# Chrome扩展手动安装指导

## 📋 安装准备
扩展文件已确认存在于：`/workspace/chrome-extension/`

## 🔧 手动安装步骤

### 第1步：打开扩展管理页面
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按Enter键访问

### 第2步：启用开发者模式
1. 查找右上角的"开发者模式"开关
2. 点击开关，确保它处于**开启状态**（应该是蓝色/高亮状态）
3. 页面应该显示更多选项

### 第3步：加载扩展
1. 点击"加载已解压的扩展程序"按钮
2. 在文件选择对话框中，导航到完整路径：`/workspace/chrome-extension/`
3. 选择整个 `chrome-extension` 文件夹
4. 点击"选择文件夹"按钮

### 第4步：验证安装
扩展安装成功后，您应该看到：
- ✅ "智能页面总结器"扩展出现在扩展列表中
- ✅ 状态显示为"已启用"
- ✅ 浏览器工具栏出现扩展图标

## ❗ 常见问题
如果遇到问题：
- 确认选择了正确的文件夹（chrome-extension）
- 确认开发者模式已正确开启
- 刷新扩展管理页面重试
