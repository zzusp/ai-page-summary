## 技术分析

### 安全限制
Chrome扩展管理页面使用了沙盒安全机制，阻止了自动化工具的正常操作。这是浏览器的安全特性，用于防止恶意扩展的自动安装。

### 替代解决方案
1. **手动安装（推荐）**
   - 用户需要手动在Chrome中打开 `chrome://extensions/`
   - 开启开发者模式（右上角开关）
   - 点击"加载已解压的扩展程序"
   - 选择 `/workspace/chrome-extension/` 目录

2. **开发者工具方法**
   - 在扩展管理页面按F12打开开发者工具
   - 在控制台中执行JavaScript代码
   - 使用 `document.querySelector()` 等方法直接操作DOM元素
