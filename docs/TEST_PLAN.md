# 智能页面总结器 - 功能测试计划

## 🎯 测试目标
验证Chrome扩展在sina.com.cn网站上的总结功能

## 📋 测试步骤

### 阶段1：基础功能验证
1. **确认安装成功**
   - 扩展图标出现在工具栏
   - 点击图标显示弹窗界面

2. **API配置测试**
   - 打开设置界面
   - 配置API密钥（OpenAI/Anthropic）
   - 测试API连接状态

### 阶段2：sina.com.cn网站测试
1. **无重点总结测试**
   - 访问 https://www.sina.com.cn/
   - 点击扩展图标
   - 点击"开始总结"（不输入任何重点）
   - 记录总结结果质量

2. **有重点总结测试**
   - 访问新闻文章页面
   - 点击扩展图标
   - 输入测试重点："新闻发布时间和主要内容"
   - 点击"开始总结"
   - 记录总结结果

### 阶段3：高级功能测试
1. **快捷键测试**
   - 使用 Ctrl+Shift+S 启动总结
   - 验证快捷键响应

2. **右键菜单测试**
   - 在页面上右键选择"总结此页面"
   - 验证上下文菜单功能

### 阶段4：设置功能测试
1. **跨设备同步**
   - 打开设置界面
   - 检查同步选项状态
   - 验证同步功能可用性

2. **参数配置**
   - 测试创造性参数调整
   - 验证不同API提供商切换

## 📸 截图记录点
- 扩展安装成功界面
- sina.com.cn网站截图
- 总结结果弹窗
- 设置界面
- 各种测试状态的记录

## 📊 测试评估标准
- ✅ 安装成功且无错误
- ✅ 扩展图标正常显示
- ✅ 总结功能正常工作
- ✅ API配置成功
- ✅ 设置功能完整
- ✅ 同步功能可用
